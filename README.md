# QuizSurvivor

> **데이콘 온라인 게임 개발 챌린지 출품작**

최후의 1인이 살아남는 한국어 서바이벌 퀴즈 게임입니다.

---

## 게임 소개

QuizSurvivor는 6~8명의 플레이어(플레이어 1명 + AI)가 라운드마다 퀴즈를 풀고, 오답 누적 횟수가 가장 많은 플레이어가 탈락하는 서바이벌 형식의 퀴즈 게임입니다.

매 라운드 시작 시 룰렛으로 카테고리가 무작위로 선정되며, 3문제씩 풀어 오답이 가장 많은 플레이어를 탈락시킵니다. 동점자가 발생하면 1:1 결투를 통해 탈락자를 결정하며, 최후의 1인이 남을 때까지 게임이 이어집니다.

---

## 게임 방법

1. **설정** — 인원 수(6~8명)와 난이도를 선택합니다.
2. **시작** — 게임 시작 버튼을 누르면 AI 플레이어가 자동으로 구성됩니다.
3. **카테고리 룰렛** — 라운드마다 룰렛이 돌아가며 퀴즈 카테고리가 무작위로 결정됩니다.
4. **퀴즈 풀기** — 제한 시간 안에 정답을 입력(또는 O/X 선택)합니다.
   - 정답 확인 후 결과 오버레이가 표시됩니다.
   - 3문제를 모두 풀면 라운드 결과 화면으로 이동합니다.
5. **탈락** — 오답 누적이 가장 많은 플레이어가 탈락합니다. 동점이면 결투(타이브레이커) 1문제로 승부를 가립니다.
6. **승리** — 최후의 1인으로 살아남으면 승리, 탈락하면 패배입니다.

---

## 퀴즈 카테고리

| 카테고리 | 아이콘 | 설명 |
|---------|--------|------|
| **노래 퀴즈** | 🎵 | K-POP 가사의 빈칸을 채우거나 노래 제목을 맞히는 퀴즈 (아이유, BTS, 빅뱅 등) |
| **OX 퀴즈** | ⭕ | 과학·상식·역사 관련 문장이 참인지 거짓인지 O/X로 선택하는 퀴즈 |
| **영화 퀴즈** | 🎬 | 줄거리 힌트와 초성을 보고 한국 영화 제목을 맞히는 퀴즈 (기생충, 부산행 등) |
| **수도 퀴즈** | 🌍 | 세계 각국의 수도를 맞히는 퀴즈 (함정 포함, 초성 힌트 제공) |
| **속담 퀴즈** | 📜 | 한국 속담의 빈칸에 알맞은 말을 채우는 퀴즈 |

> 같은 카테고리가 연속으로 나오지 않도록 자동 조정됩니다.

## 난이도

| 난이도 | 아이콘 | 제한 시간 | AI 오답 확률 |
|--------|--------|-----------|-------------|
| **쉬움** | 🌱 | 25초 | 30% |
| **보통** | ⚡ | 15초 | 15% |
| **어려움** | 🔥 | 10초 | 5% |

- **제한 시간** 내에 답을 입력하지 않으면 자동으로 오답 처리됩니다.
- **AI 오답 확률**이 낮을수록 AI가 정답을 더 많이 맞히므로, 어려움 난이도에서는 AI와의 경쟁이 치열해집니다.

## 기술 스택

- **HTML5** — 단일 파일 구조 (`index.html`)
- **CSS3** — 커스텀 CSS + Tailwind CSS (CDN)
- **Vanilla JavaScript** — 프레임워크 없이 순수 JS로 게임 로직 구현
- **Google Fonts** — Plus Jakarta Sans
